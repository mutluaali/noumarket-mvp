# UI + Category Update

Bu pakette yapılan ana değişiklikler:

- Navbar arama alanı gerçek input haline getirildi; Enter veya Ara butonu ana aramayı çalıştırır.
- Ana sayfaya sahibinden tarzı ana kategori + alt kategori bölümü eklendi.
- Alt kategori seçimi liste sonuçlarını filtreler.
- İlan detay sayfası zenginleştirildi: breadcrumb, hızlı özet, daha iyi görsel galeri, daha düzenli satıcı ve güven alanı.
- Next image quality uyarısı için next.config.mjs içine `qualities: [72, 75, 80]` eklendi.

Kurulum:

```bash
npm install
npm run dev
```

Açılış:

```text
http://localhost:3000
```
