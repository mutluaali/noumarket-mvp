# NouMarket MVP

## Local çalıştırma
```bash
npm install
npm run dev
```

## .env.local
`.env.example` dosyasını kopyalayıp `.env.local` yapın ve Supabase bilgilerini girin.

```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```
