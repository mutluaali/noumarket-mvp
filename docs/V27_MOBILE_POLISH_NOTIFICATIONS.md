# V27 — Mobile polish & notification center

Bu sürüm mobil kullanım hissini ve bildirim erişimini güçlendirir.

## Gelenler

- Mobil alt menüde bildirim merkezi
- Okunmamış bildirim badge'i
- Mesaj ve ilan durum bildirim trigger'ları
- Notification tablo indeksleri
- Mobil kullanım için daha hızlı erişim akışı

```bash
npm install
npm run dev
```

git add .
