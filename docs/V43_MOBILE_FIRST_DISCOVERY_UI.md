# V43 - Mobile-first discovery UI

Bu güncelleme ana sayfayı daha sade, hızlı algılanan ve mobil-first marketplace deneyimine yaklaştırır.

## Değişiklikler

- Header sadeleştirildi.
- Masaüstü header içine hızlı arama tetikleyici eklendi.
- Mobil header daha kompakt hale getirildi.
- Ana hero alanı daha net mesaj ve daha az görsel gürültü ile düzenlendi.
- İstatistik kartları sadeleştirildi.
- Mobilde filtre erişimi üst alana alındı.
- Detaylı filtre bloğu daha kısa ve okunaklı hale getirildi.
- İlan kartları sadeleştirildi; görsel, fiyat, başlık ve konum öne çıkarıldı.
- Kartlardaki fazla ikon/metin yükü azaltıldı.

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıdan aç:

```text
http://localhost:3000
```
